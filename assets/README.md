# Assets

This folder stores lightweight assets used by the repository documentation.

Current contents:

- `teaser.svg`: a simple repository teaser graphic for the README.

Suggested future additions:

- paper teaser figure
- benchmark snapshots
- side-by-side qualitative comparisons
- short GIF or MP4 previews linked from the README

Large videos and checkpoints should not be committed directly to git. Host them externally and link to them from the README or the project page.
